# Matching-Card-Game

## Description:
 This is a browser based, retro styled memory game. it tests how sharp your memory is, it tells you how much moves it took you to complete the game.

## How it works:
 When you press play, javascript shuffles 18 pairs of emojis and drops them into the grids. Click any card to trigger a smooth flip effect. The logic checks two cards at a time if they match, they pop and lock in place, if they dont the board pauses for a second so you can memorize them before they flip back down. Once you clear all 18 pairs, a winning screen pops up with your final move count.

## Why i built this:
 I wanted to build something fun and visual to sharpen my Javascript skills. I really wanted to get comfortable managing game states, preventing user click spamming while cards are animating, and handling dynamic DOM Elements,plus i got to experiment with CSS grid layout and custom keyframe animation to create that arcade vibe.

## Tech Stack:
 * HTML: layouts for the start, game, and winning scrrens.
 * CSS: heavy borders, Card flips. and responsive grid architecture.
 * JS: state tracking, array shuffling, and real time DOM Updates.
